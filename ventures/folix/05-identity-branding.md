## 3. Elimination Audit Trail
| Name | Code | Evidence / Logic |
| :--- | :---: | :--- |
| **Vink** | [TM] | Direct collision: *vink.com.mx* (Logistics Tech in Mexico City). High risk of confusion. |
| **Enlace** | [TM] | Direct collision: *Enlace TP* (Mexican Payment Processor). Market leader. |
| **Lexis** | [PHON] | Near-miss collision: *LexisNexis*. Brand shadowing risk. |
| **Fiscalia** | [NAR] | Sounds like "Government Prosecutor." Too aggressive/threatening for an assistant. |
| **Nexu** | [TM] | Direct collision: *Nexu.mx* (Mexican automotive fintech). |
| **Signa** | [TM] | Direct collision: *Signa.so* (API platform). |
| **Folio** | [STRAT] | Too descriptive. "The Folio" is a generic term in MX fiscal law; hard to own. |
| **Signify** | [TM] | Direct collision: *Signify* (Philips lighting global brand). |
| **Vault** | [STRAT] | Too generic. Hard to own SEO and Trademark for a software category word. |
| **Bridge** | [STRAT] | Too generic. Hard to own. |
| **Linkr** | [DOM] | Highly squatted handles and .com priced as premium. |
| **Nexf** | [PHON] | Hard to say in a noisy room (Sibilance cluster at end). |

## 4. Marketing & Brand Strategy (Leading Candidate: Folix)
### 4.1 Tagline Options
1.  **"Tus folios, bajo control."** (Authority-based)
2.  **"La brújula de tu facturación."** (Navigation-based)
3.  **"Fiscalidad sin fricción."** (Speed-based)

### 4.2 Visual Identity Direction
*   **Palette:** Electric Cobalt (#2563EB) for Trust + Slate-900 (#0F172A) for Foundation.
*   **Typography:** Clean, high-contrast grotesque (e.g., *Inter*) - Medium weight for maximum readability.

### 4.3 Domain Strategy
*   **Primary:** `folix.mx` (Verified Available).
*   **Secondary:** `folix.io` (Verified Available).

### 4.4 Brand Hierarchy
*   **Folix Vault:** The Identity/CSD storage product.
*   **Folix Bridge:** The webhook/payload ingestion engine.
